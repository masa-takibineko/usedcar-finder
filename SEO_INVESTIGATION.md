# 検索表示調査メモ（2026-02-28）

## 事象
- 「中古車オークション代行」で検索しても見つかりにくい
- 「丸印モーター」検索では「手数料」ページが目立つ

## コードベース上で確認できた事実
1. **インデックスブロック設定は見当たらない**
   - `robots` は `index: true / follow: true`
   - `robots.txt` は `Allow: /`（`/data/`のみ拒否）
2. **Search ConsoleのHTML確認ファイルは配置済み**
   - `public/googlece9988ac3252f2f1.html` が存在
3. **サイトマップは存在**
   - `public/sitemap.xml` に主要ページ8件を掲載
4. **ホームの主要キーワード整合に弱さがあった**
   - 旧タイトル: 「中古車購入を安くするならオークション代行検索丸印モーターへ」
   - 「中古車オークション代行」という検索語に対して、タイトル/H1の完全一致が弱かった
5. **ブランド検索で手数料ページが強くなる要因**
   - `/price` の title が「オークション代行手数料 | 丸印モーター」で、
     ブランド＋検索意図（手数料）に強く一致している

## 今回の改善
- ホームページの title/description を「丸印モーター」「中古車オークション代行」に明確一致する文面へ更新
- ホームの `WebPage` JSON-LD 名称も同じ文面に統一
- ホーム本文の H1 と導入文に「中古車オークション代行」を明示

## Search Console 側で確認してほしい点
1. **登録プロパティが実ドメインと一致しているか**
   - `https://usedcar-finder.vercel.app/` で登録しているか
2. **URL検査でホーム(`/`)が「インデックス登録済み」か**
3. **「重複・正規URL」の判定先が他URLになっていないか**
4. **サイトマップ送信先が同一プロパティ配下か**
5. **手動対策・セキュリティ問題がゼロか**

## 補足
- このリポジトリ確認だけでは、実際のSearch Console管理画面（所有権・カバレッジ・クロール統計）までは断定できない。
- ただし、コード上は「未登録だから絶対に出ない」というより、
  **ホームの検索語一致と評価蓄積の弱さ**が主因である可能性が高い。
